cursor movement-------
w : forawrds (start)
W : forwards (start(space))
e : forwards (end)
E : forwards (end(space))
b : backwards
B : backwards
$ : end of line
0 : start of line

insert-------
ctrl h : delete character (before)
ctrl w : delete word (before)
ctrl t : tab current line
ctrl j : new line
editing-------
R : replace many characters
ctrl r : redo

cut and paste-----
yy : copy a line
yw : copy character
p : paste (after)
P : paste (before)
